## Описание

Три SDD-скилла — apply, contradiction, archive — получили финальную фазу write-then-replay: форматированный пользовательский отчёт теперь записывается в отдельный файл и воспроизводится через `cat`. Это решает проблему потери вывода при запуске скиллов внутри Agent tool. Параллельно добавлено правило в `claude-way.md`, фиксирующее паттерн как стандарт для всех скиллов с многосекционным финальным выводом.

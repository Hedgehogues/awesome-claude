## Requirements
### Requirement: ## Что делать содержит только человекочитаемые задачи
Секция `## Что делать` в диалоговом выводе скиллов `sdd:contradiction`, `sdd:apply`, `sdd:propose`, `sdd:archive` SHALL содержать только строки вида `- [ ] <действие>`, где действие описано на русском языке без упоминания имён файлов, имён детекторов, технических идентификаторов. Каждая строка SHALL быть достаточной для принятия решения без дополнительного контекста.

#### Scenario: Действие описано без файловых путей
- **WHEN** contradiction находит redundancy-предупреждение для design.md:3
- **THEN** в `## Что делать` появляется строка вроде `- [ ] Убери дублирующийся мотивационный абзац из раздела «Context» — он уже есть в тексте предложения`
- **THEN** строка НЕ содержит `design.md`, `pointer-rewrite`, `redundancy`, `SSOT`

#### Scenario: Действие достаточно для команды Клоду
- **WHEN** пользователь читает `## Что делать` без открытия других файлов
- **THEN** каждая строка понятна без дополнительного контекста
- **THEN** пользователь может передать строку Клоду как инструкцию («сделай первый пункт»)

### Requirement: ## Вопросы к пользователю не требует знания структуры файлов
Секция `## Вопросы к пользователю` SHALL формулировать вопросы в терминах задачи и намерения, а не в терминах структуры файлов и технических артефактов.

#### Scenario: Вопрос о выборе SSOT сформулирован по-человечески
- **WHEN** contradiction обнаруживает decision-gate с выбором SSOT
- **THEN** вопрос звучит как «Где хранить основной текст мотивации — в тексте предложения или в документе с техническими решениями?»
- **THEN** вопрос НЕ содержит «proposal.md», «design.md», «SSOT», «pointer-rewrite»

### Requirement: Диалоговый вывод содержит ссылку на лог-файл
После основного вывода скилла SHALL присутствовать одна строка-ссылка на лог-файл текущего запуска: `→ Подробный технический отчёт: .log/<skill>-<timestamp>.md`

#### Scenario: Ссылка на лог добавлена в конец вывода
- **WHEN** скилл завершает работу и вывел пользовательские секции
- **THEN** последняя строка перед `## Вопросы к пользователю` — ссылка на лог-файл
- **THEN** путь в ссылке относительный (от корня репозитория)

# DB Policies

# Контекст
Каталог содержит нормы проектирования и эксплуатации БД для проекта: write/read модели, ограничения целостности, миграции, безопасность, производительность и ретенцию.

## Структура
- CONSTRAINTS.md — ограничения БД и инварианты (PK/FK/UNIQUE/CHECK/NOT NULL).
- INDEXES.md — индексация, планы запросов, запреты на N+1.
- MIGRATIONS.md — правила миграций схемы и данных, безопасность и порядок.
- NORMAL_FORMS.md — нормы нормализации write-модели и исключения.
- PERFORMANCE.md — бюджеты и требования производительности.
- READ_MODEL.md — правила витрин и проекций для чтения.
- RETENTION.md — хранение, удаление и архивирование данных.
- SECURITY.md — безопасность доступа, роли и PII.
- SEEDS_FIXTURES.md — seed-данные и фикстуры.
- TRANSACTIONS.md — транзакции и outbox для событий.
- VERSIONING.md — версионирование агрегатов, схемы и событий.
- WRITE_MODEL.md — правила write-модели и границ агрегатов.

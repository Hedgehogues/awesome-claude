# CODEBASE.md — кодовая база

# Контекст
Документ описывает правила организации кодовой базы и веток деплоя. Цель — единый источник истины и предсказуемое развертывание в разных окружениях.

## Правила
- Один репозиторий = одна кодовая база.
- Одна кодовая база разворачивается в разных окружениях через конфигурацию.
- Любое окружение должно быть воспроизводимо из репозитория.
- Экспериментальные изменения живут в отдельных ветках и не меняют основную историю.
- Зависимости общие, окружение задается параметрами, а не отдельными ветками.
- Сборочные артефакты не хранятся в репозитории.

## Антипаттерны (запрещено)
- Разные форки для разных окружений.
- Ручные правки прод-кода без фиксации в репозитории.

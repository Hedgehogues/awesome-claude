# Practice

## Контекст
В этом каталоге собраны группы запахов кода, которые указывают на проблемы в
структуре, связности и дизайне. Эти материалы используются для диагностики
проблем и выбора подходящих рефакторингов.

## Содержимое
- `CHANGE_BREAKERS.md` — запахи, делающие изменения дорогими.
- `DEPS.md` — запахи, связанные с зависимостями и связностью.
- `INFLATORS.md` — запахи раздувания методов и классов.
- `OOP_DESIGN.md` — запахи неправильного использования OOP.
- `TRASHERS.md` — запахи «мусора» и лишних элементов.

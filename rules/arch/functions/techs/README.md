# Techs

## Контекст
Каталог содержит группы рефакторингов и приемов, которые применяются для
упрощения логики, управления данными, улучшения иерархий и повышения
читаемости кода. Раздел служит справочником техник, применимых к разным
типам проблем.

## Содержимое
- `CONDITIONS.md` — рефакторинги условной логики.
- `DATA.md` — рефакторинги работы с данными.
- `FUNCTIONS.md` — перемещение функциональности между классами.
- `GENERALIZATIONS.md` — рефакторинги для наследования и обобщений.
- `METHODS.md` — составление и разбиение методов.
- `SIMPLIFY.md` — упрощение интерфейсов методов.
